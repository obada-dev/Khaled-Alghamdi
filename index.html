Project 01 · Embedded Systems + IoT
Home Controllable Humidifier

Designed and implemented an automated home humidifier system with smart environmental monitoring. Uses a DHT11 sensor for real-time temperature and humidity readings displayed on an LCD. Users set desired humidity via a potentiometer; the system automatically activates an ultrasonic mist maker and fan when humidity drops below threshold. A water-level sensor provides automatic shutdown protection to prevent hardware damage.

Arduino DHT11 Sensor LCD Display Relay Module Ultrasonic Mist Water Level Sensor C++

Project 02 · Embedded Systems + Sensors
Smart Garage Side Parking Sensor

Designed and implemented an automated parking assist system using Arduino and an HC-SR04 ultrasonic sensor to measure vehicle-to-wall distance in real-time. Features a 4-digit 7-segment display for accurate distance readout, with dynamic RGB LED and Buzzer feedback alerts based on proximity thresholds (<10cm, 10–30cm, >30cm) to prevent collisions and enhance safety.

Arduino HC-SR04 Ultrasonic 7-Segment Display RGB LED Buzzer C++
